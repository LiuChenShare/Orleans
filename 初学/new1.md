# Microsoft Orleans 快速入门
Microsoft Orleans 是由微软研发的一个在.net下用简单方法构建高并发、分布式的大型应用程序框架。  
官方文档以及源码：http://dotnet.github.io/orleans/  
Orleans 框架可以构建大规模、高并发、分布式应用程序，而不需要学习专业分布式以及并发知识。  
这篇随笔主要记录了自己学习Orleans的经过和理解，在学习过程中会一直更新，思路和理解可能有些偏颇，如果有幸有大佬看到这篇文章，希望能给予批评指正。
